# 💰 MoneyUp

Aplicação web de controle de gastos utilizando:

- Node.js
- Express
- SQLite
- HTML
- CSS
- JavaScript Modular (ESM)

# Funcionalidades

Cadastro de gastos  
Listagem de gastos  
Exclusão de gastos  
Integração Front-end e Back-end  
CRUD completo  

# Estrutura

moneyup/
backend/
frontend/
package.json
README.md

# Como executar

## Instalar dependências

npm install

## Rodar servidor

npm start

## Abrir front-end

frontend/index.html
